Acerca del Proyecto

Este proyecto tiene como objetivo:

  • Aplicar el framework Django para crear una aplicación web funcional.
  • Implementar vistas, rutas y el uso de URLs con templates.
  • Agregar estilos modernos utilizando Tailwind CSS.
  • Aprender estructura MVC y organización profesional de proyectos web.
